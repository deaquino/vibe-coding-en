# Contributing to This Project

Thank you for your interest in this project! We welcome all forms of contributions.

## How to Contribute

-   **Report Bugs**: If you find any errors or unclear sections while reading or using the guide, please submit your issue through the [Issues](https://github.com/tukuaiai/vibe-coding-cn/issues) page. Please describe the problem in as much detail as possible.
-   **Feature Suggestions**: If you have any suggestions for improving this guide, please let us know through the [Issues](https://github.com/tukuaiai/vibe-coding-cn/issues) page.

## Submitting Changes (Pull Request)

If you would like to contribute content directly:
1.  Fork this repository.
2.  Create a new branch (`git checkout -b feature/YourAmazingFeature`).
3.  Make your modifications.
4.  Commit your changes (`git commit -m 'feat: Add some AmazingFeature'`).
5.  Push the branch to your Fork (`git push origin feature/YourAmazingFeature`).
6.  Create a new Pull Request.
